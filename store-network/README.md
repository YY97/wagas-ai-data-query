# 门店网络效率诊断地图（Store Network Efficiency Map）

基于 Wagas 集团门店**地理位置**与**店日均销售额（ADS）**的交互式地图可视化工具。

## 核心功能

- 已开业门店地图打点（高德底图）
- **1km 覆盖圈** + 重合度分析（高亮多店重叠区域）
- **店日均销售额（ADS）分段着色**：
  - `< ¥5,000` / `¥5,000–10,000` / `¥10,000–20,000` / `> ¥20,000`
- 多维筛选：品牌、城市、ADS 区间、门店名搜索
- 弹窗明细：门店信息、ADS、1km 内重合门店逐家列出

## 分析框架：门店网络效率诊断

| 维度 | 回答的问题 |
|------|-----------|
| **密度效率（蚕食分析）** | 这个商圈是不是店开太多了？重合度高时单店 ADS 是否被稀释？ |
| **空间效率（城市对标）** | 同城同品类，为什么这家店比那家差？如何设定同类店 ADS 基线？ |
| **空白效率（选址决策）** | 哪里 ADS 高但尚无同品牌竞争 → 优先新开；哪里全是低 ADS → 不新开 |
| **组合效率（品牌策略）** | Wagas / Baker & Spice / Lokal 共存是互补还是互冲？ |

## 数据来源

- 门店经纬度：`stores_all.csv`（BI StoreMaster Report，筛选 `门店状态 = 已开业`）
- 店日均销售额：FIN BI 本期 ADS 数据
- 仅展示**同时具备经纬度与 ADS** 的门店（349 家）

## 生成方式

`gen_coverage_v2.py` 读取 Excel ADS 数据与 CSV 门店数据，计算 1km 内重合门店，
输出 `index.html`（单文件，依赖 CDN：Leaflet + 高德瓦片）。

```bash
pip install pandas
python gen_coverage_v2.py
```
